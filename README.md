# KevioFocus - Frontend

Interface web do **KevioFocus**, um aplicativo de produtividade baseado na técnica Pomodoro. Desenvolvido com Angular 21, consome a API REST do backend em .NET.

## Sobre o projeto

O KevioFocus permite criar sessões de foco personalizadas com duração, pausas e ciclos configuráveis. Durante uma sessão, o usuário acompanha um timer visual em tempo real, gerencia suas tarefas e registra o progresso de cada execução.

## Funcionalidades

- Criação e listagem de sessões de foco
- Timer Pomodoro com foco, pausa e ciclos automáticos
- Gerenciamento de tarefas por sessão (criar, concluir, remover)
